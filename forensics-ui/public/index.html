What This Tool Does

  • Forensic Extraction: We capture the complete "digital fingerprint" of X posts. This includes not just the text, but full metadata, server headers, and raw JSON content often lost in screenshots.
  • Blockchain Anchoring: To ensure immutability, we generate a SHA-256 hash of all evidence artifacts and anchor them to the Ethereum/Polygon blockchain. This creates a permanent, third-party proof-of-existence.
  • AI Defamation Analysis: Using advanced NLP models, ChainForensix scans captured posts for patterns of defamation, hate speech, and coordinated harassment, providing objective scores for legal dossiers.

Technical Methodology

Our process follows strict forensic standards:

  1. Ingestion: Targeted URL acquisition via high-velocity scraping.
  2. Verification: Automatic verification of data integrity via cryptographic hashing.
  3. Persistence: Evidence anchoring to public ledgers (Blockchain).
  4. Reporting: Generation of cryptographic forensic reports in PDF and JSON formats.
