ChainForensix: Professional X (Twitter) Forensics

The only forensic tool combining high-velocity X post extraction with cryptographic blockchain anchoring.

What This Tool Does

  • Forensic Extraction: Captures the full "digital fingerprint" of X posts, including metadata, server headers, and JSON content.
  • Blockchain Anchoring: Generates a SHA-256 hash of the evidence and anchors it to the Ethereum/Polygon blockchain for immutable proof-of-existence.
  • AI Defamation Analysis: Uses advanced NLP models to scan posts for defamation, hate speech, and harassment patterns.

Who It Is For

Justice Seekers

Victims of online harassment seeking court-admissible evidence.

Investigators

OSINT professionals requiring verified chain-of-custody.

Legal Counsel

Attorneys needing technical proof for defamation cases.

How It Works

  1. Input: User provides an X post URL.
  2. Process: Engine captures the post + metadata and analyzes text with AI.
  3. Anchor: Evidence hash is written to the blockchain.
  4. Output: A signed PDF/JSON report is generated.
